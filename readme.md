## Описание:
Данный бот был создан для практики работы с библиотеками asyncio, aiogram и aiosqlite

Реализован в формате квиза по Python с четырьмя вариантами ответов

## Команды:
### /start:
Стандартная команда telegram при запуске бота, после ее отправки выводится приветствие и создаются две кнопки ReplyKeyboardBuilder начать игру и посмотреть статистику

### Начать игру или /quiz
Выводит сообщение, о том что квиз начинается, создает или перезаписывает пользователя по его TelegramID с нулевым ID вопроса и количеством очков, и выводит первый вопрос

### ans: {answer_id}
Считывает ID ответа пользователя, проверяет его корректность, выводит сообщение, о том, правильно ли ответил пользователь, и добавляет очки за верный ответ. Так же в данной функции производится проверка, на последний вопрос квиза, если вопрос последний, выводится уведомление, о том что квиз окончен.

### Посмотреть статистику
Выводит статистику по участникам из базы данных

## P.S
API-ключ для бота добавлен в .env, и вытаскивается с помощью функции в ./config/settings.py
